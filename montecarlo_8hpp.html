int monte_carlo ( )

Monte Carlo method for predicting the number representing pi TODO.

A Monte Carlo method is really a problem dealing with random distribution and random sampling related technique. In this case and within this file we use a "Monte Carlo Method" from what is talked about in the video below specifically when the author talks about predicting the number pi π using random sampling by "dropping marbles" into a square and determining how many are dropped within 1/4 of the square. The number of successes divided by the number of total attempts will our prediction of pi π
